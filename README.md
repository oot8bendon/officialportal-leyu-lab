# officialportal-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常作为临时展示、测试或文档用途，并非绑定于某个固定站点或域名。

## 项目简介

officialportal-leyu-lab 是一个轻量的页面归档仓库。  
其中存放的 HTML 文件各自独立，可直接通过浏览器打开查看，或部署到静态托管服务中用于快速演示与内容分发。

## 目录结构说明

```
.
├── index.html          # 仓库入口页面（可选）
├── page1/              # 示例页面目录
│   └── index.html
├── page2/
│   └── index.html
└── README.md           # 本文件
```

目录结构可能随时调整，请以仓库实际内容为准。

## 页面归档说明

- 所有 HTML 页面均为独立文件，不依赖后端服务。
- 页面内可能包含简单的样式与脚本，均为前端静态资源。
- 部分页面可能为临时性内容，会随着仓库更新而增删。

## 维护说明

- 本仓库由维护者不定期更新，主要用于个人或团队内部归档。
- 不保证页面内容长期可用或兼容所有浏览器版本。
- 如有特定页面需要长期保留，建议自行备份。
